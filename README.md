# OIBSIP_python_task2
Random Password Generator using Python

## Objective
The objective of this project is to generate strong and secure random passwords based on user-defined criteria.

## Steps Performed
1. Allowed the user to choose password length.
2. Included letters, numbers, and symbols.
3. Generated a random password using Python modules.
4. Displayed the generated password to the user.

## Tools Used
- Python
- Visual Studio Code
- GitHub

## Outcome
The program generates secure and customizable passwords, helping users improve account security.
